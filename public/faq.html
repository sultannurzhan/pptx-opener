Frequently Asked Questions

Quick answers about compatibility, behavior, and limits

Use this page for common questions around file support, rendering quality, controls, and troubleshooting.

General

Which file type is supported?

Nexuslides currently targets .pptx files (Office Open XML format).

Do I need an account?

No account is required. Open the site and drop your deck to start rendering.

Can I use keyboard shortcuts?

Yes. Use arrow keys for navigation, F for fullscreen toggle, and Esc to exit fullscreen.

Rendering and fallback

Why does a deck look different from PowerPoint?

Some advanced objects, effects, or custom components are interpreted differently by web renderers versus native Office apps.

What happens if parsing fails?

Nexuslides automatically attempts fallback mode to extract readable slide text instead of failing silently.

Is there a file size limit?

There is no hard product limit in the UI, but browser memory and device performance impact very large decks.

Need to test with your own deck?

Go back to the viewer and open a .pptx presentation locally.
